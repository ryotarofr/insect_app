## このサイトについて

`insect_app` リポジトリの設計資料を **MDX + 図 (Astro コンポーネント)** で公開する場です。素の Markdown だけでは伝わりにくい「型のフロー」「リクエストのライフサイクル」を SVG 図で補強し、コードベースの**意思決定の理由**を後から読める形で残すことを狙いとしています。

正典は `docs/sdui-three-layer-model-v6.md` (= リポジトリ内の素の md) で、本サイトはそれを取り込んでナビゲーション・サイドバー・TOC を付けた読み物バージョンとして運用します。

## このサイトの 3 つの方針

<CardGrid>
  <Card title="正典は md、サイトはその派生" icon="document">
    リポジトリ内の素の md (`docs/`) が引き続き Source of Truth。本サイトは frontmatter を被せて読みやすくしたミラーで、レビュー / LLM プロンプトでは引き続き素の md を参照します。
  </Card>
  <Card title="図は Astro コンポーネントで再利用" icon="puzzle">
    `src/components/*.astro` に SVG 図を切り出し、MDX から `import` で埋め込みます。Starlight の light/dark どちらでも視認できるよう明示色で書く規約です。
  </Card>
  <Card title="GitHub Pages へ自動デプロイ" icon="rocket">
    `main` push (= `docs-site/` の変更時) で GitHub Actions がビルド / デプロイ。コード変更とドキュメント更新を 1 つの PR にまとめられます。
  </Card>
</CardGrid>
